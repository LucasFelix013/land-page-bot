O que é bot para discord?

Um bot é um software criado por Lucas, que funciona com o Discord graças à API aberta da plataforma.
Cada bot pode contar com uma série de funcionalidades diferentes, que podem ser úteis até para quem não joga,
e procura um software mais versátil para conversas em áudio, vídeo e texto.
Um bot pode desde incluir funções sociais a adicionar opções de customização, gravar o áudio de uma conversa e mais.
